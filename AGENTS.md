# AGENTS.md - Project Guidelines

## Image Guidelines
- **ONLY** use sea/water/scuba/diving related pictures across the entire website
- **NEVER** reuse pictures that have already been added to the website
- **ALWAYS** add new, unique pictures for each new course or section
- All images must be relevant to underwater activities, marine life, diving equipment, or ocean environments
- Before adding any image, verify it has not been used elsewhere in the project
